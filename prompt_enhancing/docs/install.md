# Installation

In your environment you need the following dependencies:

- `python` (>= 3.13)
- `poetry` (>=2.1)
- `graphviz`
- `jupyter-notebook` or `jupyter-lab`, with the python package [**`poetry-kernel`**](https://github.com/pathbird/poetry-kernel)

```sh
poetry instll --all-extras
```

```sh
just run-lab
```
